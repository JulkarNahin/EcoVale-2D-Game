# EcoVale 2D Game 🌱

EcoVale is a Java‑based 2D game developed in Eclipse IDE.  
It features tile maps, interactive gameplay mechanics, and dynamic trash placement — designed as part of an advanced programming project.

---

## 🎮 Features
- 2D tile‑based world
- Interactive player controls
- Dynamic object placement
- Optimized runtime stability

---

## 📂 Project Structure
```
EcoVale/
├── src/        # Java source code
├── res/        # Game assets (images, maps, fonts)
├── .project    # Eclipse project file
├── .classpath  # Eclipse classpath file
└── EcoVale.jar # Runnable JAR (optional)
```

---

## ▶️ How to Run

### Option 1: Run from Source
1. Clone the repo:
   ```
   bash
   git clone https://github.com/JulkarNahin/EcoVale-2D-Game.git
   cd EcoVale-2D-Game
   ```
   
2. Compile the code:
   ```
   javac -d bin src/main/*.java src/entity/*.java src/objects/*.java src/tile/*.java
   ```
   
4. Run the game:
   ```
   java -cp bin main.GamePanel
   ```
   
### Option 2: Run the JAR
If you prefer not to compile:
```
java -jar EcoVale.jar
```

## 🛠 Requirements
- Java JDK 8+
- Eclipse IDE (optional, for development)
